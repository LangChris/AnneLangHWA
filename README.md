# AnneLangHWA
<img src="https://annelanghwa.com/assets/images/anne-logo.jpg" alt="HWA Logo" />
_This is an online order system for HWA generated with [Angular CLI](https://github.com/angular/angular-cli) version 9_

## Features
- [X] Plan Coverages & Pricing Table
- [X] Buyer Order Form For Plan(s) `Gold, Platinum, Diamond`
- [X] Seller Order Form For Plan(s) `Free Sellers Coverage`
- [X] HWA Brochures For Language(s) `English, Spanish`
- [X] Admin Portal 
  - View Orders Table
  - Edit Orders Table With Function(s) `Update, Delete`
  - Export to Excel
